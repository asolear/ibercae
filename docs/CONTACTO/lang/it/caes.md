# 

## Intro
I "Certificati Bianchi," o "Titoli di Efficienza Energetica" (TEE), sono strumenti introdotti in Italia per incentivare l'efficienza energetica. Consentono alle imprese di ottenere certificati in base ai risparmi energetici conseguiti attraverso interventi di efficienza energetica. 

[Sito del GSE (Gestore dei Servizi Energetici)](https://www.gse.it/servizi-per-te/efficienza-energetica/certificati-bianchi/documenti)
